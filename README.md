ArrayList에는 최근의 30개의 News의 제목을 저장하기 위한 자료구조이고, 
HashMap은 접속한 클라이언트의 ID와 Socket을 저장하기 위한 자료구조입니다.
이를 통해 귓속말을 보내거나 전체에게 공지사항을 전송시킬 수 있습니다.
